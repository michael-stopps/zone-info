# Zone-Info

A lightweight, highly customizable World of Warcraft interface addon that displays your current subzone, zone, and character coordinates cleanly at the top of your screen.

## Features

* **Live Location Data:** Accurately displays your current subzone, zone, and exact coordinates in real-time.
* **Interactive Coordinates:** 
  * Click the coordinates to automatically copy a `/way` command to your clipboard.
  * Generate native map links directly in your chat for easy sharing.
* **Fully Customizable UI:** 
  * Lockable positioning to prevent accidental drags.
  * Toggleable background visibility.
  * Text scaling to perfectly match your UI setup.
* **Performance Optimized:** Built with a lightweight update loop to ensure minimal memory usage and zero impact on framerates during intense gameplay.

## Installation

Download the latest release from the Releases page or use WowUp's `Install from URL` functionality for automatic updating.

## Usage & Configuration

Zone-Info is designed to work right out of the box, but you can configure it entirely to your liking through the in-game settings menu.

To access the configuration menu, type either of the following slash commands in chat:
* `/zi`
* `/zoneinfo`

From the menu, you can adjust scaling, lock or unlock the frame, and manage background visibility.
